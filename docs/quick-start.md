# Quick Start

## 1. Install Node.js

Download and install Node.js LTS:

```text
https://nodejs.org/
```

Close and reopen PowerShell after installation.

## 2. Run the Lite Setup Script

Open PowerShell in this folder and run:

```powershell
powershell -ExecutionPolicy Bypass -File .\OneClickDeploy-Lite.ps1
```

## 3. Paste Your DeepSeek API Key

When asked, paste your DeepSeek API key.

## 4. Start Claude Code

After setup, close and reopen PowerShell.

Go to your document folder:

```powershell
cd "$HOME\DocuFlow-AI-Documents"
```

Start Claude Code:

```powershell
claude
```

## 5. Ask It to Edit Documents

Example:

```text
Please polish the documents in this folder. Make the writing smoother but do not change the original meaning.
```
