# Basic Prompt Templates

## Polish a Document

```text
Please polish this document. Make the writing clearer, smoother, and more professional, but do not change the original meaning.
```

## Summarize Notes

```text
Please summarize this document into 5 concise bullet points.
```

## Formal Rewrite

```text
Please rewrite this document in a more formal tone while keeping the original structure.
```

## Study Notes

```text
Please turn this material into clear study notes with headings, bullet points, and key terms.
```

## Folder Overview

```text
Please scan this folder and create a README file that explains what each file is for. Do not modify any file until I approve.
```
