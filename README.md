# DocuFlow AI Lite

DocuFlow AI Lite is a lightweight Windows setup helper for creating a local AI document assistant workflow with Claude Code and an OpenAI-compatible API such as DeepSeek.

It is designed for users who want to edit, polish, summarize, and organize local documents with an AI assistant from the terminal.

## What This Lite Version Includes

- A simple Windows PowerShell setup script
- Basic Claude Code + DeepSeek API configuration
- A small set of document editing prompt templates
- A short quick start guide

## What You Can Use It For

- Polish local text documents
- Summarize notes and drafts
- Rewrite documents in a more formal tone
- Create study notes
- Organize text and Markdown files

## Requirements

- Windows 10 or Windows 11
- PowerShell
- Internet connection
- Node.js / npm
- A DeepSeek API key

## Quick Start

Clone or download this repository.

Open PowerShell in the project folder and run:

```powershell
powershell -ExecutionPolicy Bypass -File .\OneClickDeploy-Lite.ps1
```

Then follow the instructions in the terminal.
